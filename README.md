# Courses Platform

Backend API base for a corporate e-learning platform.

## Scripts
- `npm run dev`
- `npm run build`
- `npm run start`

## Environment variables
Copy `.env.example` to `.env.local` and fill in:
- `NEXT_PUBLIC_SUPABASE_URL`
- `NEXT_PUBLIC_SUPABASE_ANON_KEY`
- `SUPABASE_SERVICE_ROLE_KEY`

## API
Routes live under `/pages/api`.
